# Go Cheat Sheet

This cheat sheet is split on purpose. One file for all 11 practiced topics would be too dense to scan quickly in an IDE tab or on paper.

## Index

- [Basics](./GO-CHEATSHEET-BASICS.md)
  Open this when you need syntax for data shapes and day-to-day value handling: runes, pointers, slices, maps, structs, and constructor-like config.
- [Patterns + Runtime](./GO-CHEATSHEET-PATTERNS-RUNTIME.md)
  Open this when you need behavior and flow control patterns: interfaces, defer, helper patterns, concurrency, context, and `net/http`.
